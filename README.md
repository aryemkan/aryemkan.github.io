# United States of Liberty (USL) — РП-портал

Статический сайт государства USL для РП-проекта в Minecraft. Работает без сервера и базы — все аккаунты хранятся в браузере (localStorage).

## Возможности
- Регистрация и вход граждан
- Цифровой паспорт (баланс, работа, транспорт, недвижимость, биография)
- Гос. реестр / MDT с поиском по гражданам, авто и собственности
- Админ-панель: выдача ролей, бан/разбан, выдача денег, удаление аккаунтов

## Аккаунт администратора

- **Пароль:** `usl2024` (смени его после первого входа через редактирование — или прямо в коде `app.js`, переменная `ADMIN_DEFAULT_PASSWORD`)

## Файлы
- `index.html` — страница
- `styles.css` — стили
- `app.js` — вся логика (авторизация, паспорт, реестр, админка)
- `assets/logo.png` — герб USL

## Как залить на GitHub Pages
1. Создай репозиторий на GitHub и загрузи туда `index.html`, `styles.css`, `app.js` и папку `assets/`.
2. Открой **Settings → Pages**.
3. В разделе **Build and deployment** выбери **Source: Deploy from a branch**, ветку `main` и папку `/ (root)`.
4. Сохрани — через минуту сайт будет доступен по адресу `https://ТВОЙ-НИК.github.io/ИМЯ-РЕПО/`.

## Важно
Это РП-проект. Авторизация учебная (хэш в localStorage), не используй реальные пароли. Аккаунты привязаны к конкретному браузеру/устройству. Если нужна общая база для всех игроков с любого устройства — потребуется бэкенд (например Next.js + Neon).
